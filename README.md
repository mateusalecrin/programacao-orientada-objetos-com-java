<h1 align="center">Programação Orientada a Objetos com Java</h1>

## Exercícios

<ol>
<li>CalculadoraV1 ao V7: Uma calculadora simples com as operações de subtração, adição, multiplicação, divisão. Porém a cada versão é aplicado um conceito de POO e também de padrão de projetos, como: 
    <ul>
    <li>Interface </li>
    <li>Herança </li>
    <li>Polimorfismo </li>
    <li>Divisão das classes em pacotes no Padrão MVC (controle, modelo, visualização)</li>
    </ul>
</li>
<li>Construção Casa (Uma casa com a quantidade de janelas e portas informadas pelo usuário, tanto quanto seu estado (aberto/fechado))</li>
<li>Fábrica de Carros (Simulação de uma Fábrica real)</li>
<li>Conta Bancária (Simulação de uma conta bancária real)</li>
<li>Material de Construção (Simulação de uma Loja real)</li>
</ol>
